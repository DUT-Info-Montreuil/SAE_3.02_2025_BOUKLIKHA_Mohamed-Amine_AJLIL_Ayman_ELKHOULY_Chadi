# SAE LA BUVETTE
Projet SAE PHP – La Buvette

Ce projet a été réalisé dans le cadre d’une SAE en BUT Informatique. Il consiste à développer une application web en PHP avec une architecture MVC permettant de gérer une buvette. L’application permet de gérer les produits, les achats, les ventes ainsi que le stock disponible en temps réel.

Le système calcule automatiquement les quantités restantes à partir des entrées (achats) et des sorties (ventes), afin d’aider à la gestion des stocks. Une interface simple permet de visualiser les produits, d’effectuer des opérations et d’assurer le suivi de l’activité de la buvette.

Ce projet met en pratique l’utilisation de PHP, SQL, MySQL, HTML/CSS et l’organisation du code avec un modèle clair séparant la logique, l’affichage et le contrôle.
